## Reporting a Vulnerability

We take the security of **HostelHub** seriously. If you find any security vulnerability, please do **NOT** open a public issue.

### How to Report?

1.  Please email the details of the vulnerability to **kritg0160@gmail.com** 
2.  Include the steps to reproduce the vulnerability.
3.  We will acknowledge your report within 48 hours.

### Policy

*   We will investigate and fix legitimate security issues promptly.
*   We request that you allow us reasonable time to fix the issue before making it public.
*   We will provide full credit to reporters of valid security issues.
